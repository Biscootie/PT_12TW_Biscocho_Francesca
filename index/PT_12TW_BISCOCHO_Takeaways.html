What I learnt, experienced, and opinions on:

HTML

CSS

JAVASCRIPT

When trying out this language, at first I was overwhelmed with
how much I have to memorize but I was eventually able to
get used to it. I had an easy yet tedious experience with using
HTML. Though using this language solely restricted my creativity
a bit.

Using CSS also took some time getting used to but once I found
my groove and style in CSS coding, I was able to liberally
customize things I want and arrange certain elements how I want.
I really enjoyed learning this language since I like the creative
freedom it allows and just seeing how it brings life to a site.

Learning Javascript personally was honestly difficult. I found
myself being confused constantly and although I get the logic
behind a certain line of code, I just cannot apply it when its
time to code using this language. I did not enjoy coding with this
language since I'm more of a design and colors type of person.
